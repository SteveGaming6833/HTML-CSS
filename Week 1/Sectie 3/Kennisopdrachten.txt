7. Een selector is het eerste stukje van een CSS regel.
Het vertelt welke html elementen de CSS properties krijgen die in de declaratieblok worden gezet.
8. Het declaratieblok is het code die in de accolades staat.
Het wordt gebruikt om daarmee alle html elementen in de selector de stijl te geven.
9. h5 en h6 in dezelfde selector zetten en in de declaratieblok daarvan moet het kleur worden gezet.
10. Het zorgt ervoor dat in een tekst alle zinnen in horizontale visie even lang zijn.
11. Een class is een selector waarin HTML elementen die geselecteerd zijn de CSS-stijl krijgen die in de declaratieblok worden gezet.
12. Een id is een selector waarin één HTML element die geselecteerd wordt de CSS-stijl krijgt die in de declaratieblok wordt gezet.
13. Een id is hetzelfde als een class, maar alleen één HTML element kan geselecteerd worden.
14. Margin is de ruimte tussen ruimtes. Padding is de transparante ruimte om de content heen in een box.
16. Block elements gebruiken de volle wijdte van de webpagina en inline elements doen dat niet.
